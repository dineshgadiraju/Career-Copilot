# AI Career Copilot API Documentation

## Main Backend URL

https://ai-career-copilot-backend-yzm9.onrender.com

## Main Routes

- POST /register — Create user account
- POST /login — Login and get JWT token
- GET /profile — Get logged-in user profile
- POST /resume/upload — Upload resume PDF
- GET /dashboard — Get resume score and skills
- GET /roadmap — Get career roadmap
- GET /role-recommendation — Get recommended role
- POST /chat — AI career coach
- POST /jobs/fetch-role-based — Fetch jobs based on resume role
- GET /jobs/live-recommended — Get matched live jobs
- POST /saved-jobs — Save a job
- GET /saved-jobs — View saved jobs
